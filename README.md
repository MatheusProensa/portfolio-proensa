# Proensa Portfolio

Portfólio pessoal desenvolvido por Matheus Proensa.

## Sobre

Projeto criado com foco em identidade visual, design moderno e desenvolvimento front-end utilizando React + TypeScript.

O objetivo do portfólio é apresentar projetos de branding, design gráfico e desenvolvimento web com uma estética premium, tecnológica e minimalista.

## Tecnologias

- React
- TypeScript
- Vite
- CSS
- React Icons
